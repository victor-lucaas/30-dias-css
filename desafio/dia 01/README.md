# Meu desafio: 30 dias de CSS

Este repositorio e o meu compromisso de praticar CSS todos os dias durante 30 dias.

A ideia e simples: todo dia eu vou pegar um desafio, estudar como ele funciona, mexer no codigo, deixar com a minha cara e postar no GitHub. Cada projeto vai ter HTML, CSS e uma pequena explicacao do que eu fiz e do que aprendi.

Tambem quero usar esse projeto para evoluir de verdade. Minha meta e estudar cada desafio, entender o codigo, testar alteracoes e conseguir explicar com minhas proprias palavras o que eu fiz em cada dia.

## O que eu quero fazer

- Praticar HTML e CSS todos os dias.
- Melhorar minha criatividade visual.
- Aprender efeitos, animacoes, hover, transicoes e responsividade.
- Criar o habito de postar meu progresso no GitHub.
- Transformar cada desafio em algo mais pessoal, sem apenas copiar o codigo original.
- Aprender a resolver mais coisas sozinho com pratica e constancia.
- Criar um registro real da minha evolucao como dev.

## Como vou postar

Cada dia fica dentro da pasta `Desafios/`.

Exemplo:

```text
Desafios/
  dia 01/
    index.html
    style.css
    README.md
```

Quando eu concluir um desafio, vou fazer um commit com o dia correspondente:

```powershell
git add "Desafios/dia 01"
git commit -m "Dia 01: icones sociais em camadas"
git push
```

## Dia 01

No primeiro dia eu trabalhei com icones de redes sociais em camadas. Mantive a ideia do efeito 3D no hover, mas alterei o visual para ficar mais moderno e com mais cara de projeto pessoal.

O que eu fiz no Dia 01:

- Reescrevi o HTML para ter uma apresentacao do desafio.
- Organizei os icones com classes mais claras.
- Troquei o Google Plus por GitHub.
- Removi os textos extras da tela para deixar o visual mais limpo.
- Coloquei links reais nos icones: Facebook, Twitter, GitHub, LinkedIn e Instagram.
- Criei um fundo mais bonito com degradê escuro, brilho colorido, grade suave e painel translúcido.
- Melhorei cores, espacamentos, bordas e sombras.
- Ajustei o hover para as camadas subirem com mais suavidade.
- Adicionei responsividade para funcionar melhor em telas menores.

O que pratiquei:

- `transform`
- `skew`
- `translate`
- `transition`
- `nth-child`
- `hover`
- organizacao de HTML e CSS

## Dia 02

No segundo dia eu fiz um loader animado em formato de progresso circular. A ideia foi mostrar uma porcentagem subindo de `0%` ate `100%`, com um arco acompanhando esse carregamento.

O que eu fiz no Dia 02:

- Corrigi as informacoes principais do HTML.
- Centralizei o conteudo na tela.
- Usei um fundo escuro para destacar o loader.
- Coloquei a porcentagem no centro do circulo.
- Criei o arco circular com `conic-gradient`.
- Usei `mask` para deixar o progresso com formato de anel.
- Adicionei um brilho suave no arco.
- Fiz o arco girar continuamente.
- Usei JavaScript para atualizar o numero de `0%` ate `100%`.

O que pratiquei:

- `display: grid`
- `place-items`
- `conic-gradient`
- `mask`
- `filter`
- `border-radius`
- `animation`
- `@keyframes`
- `rotate`
- variaveis CSS
- JavaScript basico
- responsividade
